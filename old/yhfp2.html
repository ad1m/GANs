Python Primer

Let's get familiar with the python language. We will start by introducing the basic data types: integers, floats, and strings.

Integers, or more commonly called ints, are exactly what you think they are. They are positive or negative whole numbers such as -2,-3,0,1,2,3, etc.. In python we do not have to declare that something is an integer such as we have to do in Java. Python will automatically pick up the integer data type for us. Let's explore some operations we can do with integers:

We see that we have created an int stored in the variable our_int and that we can use the print statement to print out our variable. We can also use the command type to check the type of our variable. Python is also great for math. Let's do some math:

              g = a/b

We see that we can use + for addition, - for subtraction, * for multiplication, ** for exponents, and / for division. We also notice that when we do 2/3 we get 0 and not 0.666666666667 like we would expect. This is because python is doing division with integers and not floats, which we will talk about next.

If we wanted to get a solution of 0.666666666667 for 2/3 we need to use a float object. To create a float we add a dot to our number as follows:

Financial data is time series data. So, we are going to need to know how to work with and handle dates. To handle dates we need to import datetime from the datetime package. We can do some simple date manipulation as follows:

Above, we see that we have handled two different formated dates and made them into the same format. The %d stands for day, %m stands for month, and %Y stands for year. We use a / in between each d,m,Y as that is how the date is originally separated in the x and y string variables. The 00:00:00 is a timestamp in the output above. There are many useful functions in the datetime module and you can read more about them in the official python guide referenced above.

Let us now talk about python's data strucures. These are objects that contain a possibly large number of other objects. Python provides 4 main data structures:

  1. Tuple - a collection of arbitrary objects. There are only a few methods available with tuples.
  2. List - a colelction of arbitrary objects. There are many methods available with lists.
  3. Dictionary - a key-calue store object.
  4. Set - an unordered colelction object for other unique objects.

We can call our function inside our function, which we call recursion. We know that 5! = 5 x 4 x 3 x 2 x 1 = 120. Let's write a recursive function that computes the factorial of a given number. To do this we are going to need to know about if and else statements. An if statement let's us execute certain code if a contition is met, else we can execute some other code. Let's take a look at an example of this in our recursive factorial function:

def factorial(a): 
    if a < 1: 
        return 1
    else: 
        s = a*factorial(a-1)
        return s

f = factorial(5)
print f

The output is as follows:

120

We created a function called factorial that takes in a parameter a. If a is less than 1 then we return 1, since 1! = 1. Else, we return s = a*factorial(a-1). This is the recursive call. When we enter 5 in for a, we see that we have a > 1 so we move to the else clause. We then have s = 5*factorial(5-1) = 5*factorial(4). Then, factorial(4) becomes 4*factorial(3) which becomes 4*3*factorial(2) which becomes 4*3*2*1. So, we have factorial(5) = 5*factorial(4) = 5*4*3*2*1 = 120. We now see how to create functions and work with if and else statements. However, there is one other conditional called elif. We see that our number is between 100 and 150 so we print b. We do not reach the else clause at the end. The elif allows us to check multiple conditions instead of just an if and an else. Let us now move onto loops.

Loops allow us to execute a statement or group of statements multiple times. There are two kinds of loops:

  1. For Loops: Executes a sequence of statements n times,
  2. While Loops: Executes a statement until a condition is met.

Let us first take a look at an example of a for loop:

x = [1,2,3,4,5]
y = []
for item in x: 
    y.append(2*item)
print y

#We can write the same for loop above as the following
y = [2*item for item in x]
print y

The output is as follows:

[2, 4, 6, 8, 10]
[2, 4, 6, 8, 10]

We first create a list x and an empty list y. We use a for loop to iterate through the elements of x and append 2 times the element in x into a list y. We see that there are two ways to write this for loop. The second method is exactly the same as the first method but only takes up one line. When writing code, we want it to be clean. This means that we want it as short as possible, but as readable as possible. The second method is a nice concise way to write the first method.

We can also place a for loop inside a for loop. This is called nesting a for loop. Let's take a look at an example of this:

x = [1,2,3]
y = ['a','b','c']
z = []
for i in x: 
    for j in y: 
        z.append((i,j))
print z

The output is as follows:

[(1, 'a'), (1, 'b'), (1, 'c'), (2, 'a'), (2, 'b'), (2, 'c'), (3, 'a'), (3, 'b'), (3, 'c')]

What we did was take all combinations as tuples of elements in x and y and place them in the empty list z. We nested two for loops. First we obtain (1,'a') and add it to list z. Then we obtain (1,'b') and add it to list z. We then obtain (1,'c') and add it to list z. Now, we have indexed through y in the inner loop and we now index through x. We now get (2,'a') then (2,'b') then (2,'c') then (3,'a') then (3,'b') then (3,'c'). All of these tuples get appended into the list z.

Let us now take a look at an example of using a while loop:

count = 10 
y = []
while count >= 5: 
    y.append(count)
    count = count - 1
print y

The output is as follows:

[10, 9, 8, 7, 6, 5]

We set a counter integer as 10 and created an empty list y. We note that >= means greater than or equal to. While our count variable is greater than or equal to 5 we append count into y. We then reduce count by 1. Some syntax we use in loops and conditionals are as follows:

  • > is for greater than
  • < is for less than
  • == is for equals
  • >= is for greater than or equal to
  • <= is for less than or equal to

We can also combine multiple conditionals inside our while loop, which we show below:

count = 10 
y = []
while count > 0:
    if count >=5 and count <=9:
        y.append(count)
    count = count - 1
print y

The output is as follows:

[9, 8, 7, 6, 5]

We see that if our count variable becomes negative (less than zero) we stop our while loop. However, if the count variable is greater than or equal to 5 and less than or equal to 9 we append it into the list y. We can also use or in our conditionals, which is shown below:

count = 10 
y = []
while count > 0:
    if count % 2 == 0 or count ==3:
        y.append(count)
    count = count - 1
print y

The output is as follows:

[10, 8, 6, 4, 3, 2]

The % in the code above is called modulo. If x % 2 is zero then we have an even number. 3 % 2 is 1 becaues 2 goes into 3 with 1 left over and 9 % 5 is 4 because 5 goes into 9 once with 4 left over. We see that if count is even or if count is equal to 3 then we add it into the list y.

We will now introduce break and continue. If we break, we exit the loop. Continue just tells us to continue doing what we are doing. An example of using break and continue within a for loop is found below:

x = [1,2,3,4,5]
for i in x: 
    if i == 4:
        print i
        break 
    else: 
        print 'continue'
        continue

The output is as follows:

continue
continue
continue
4

We see that we iterate over x. If the element we are currently on is equal to 4, we print 4 and then exit the for loop. Else, we print continue and move onto the next element. We see that we print continue, continue, continue, 4, and then end. We do not print another continue statement for element 5 in the for loop because once we hit element 4 we break and leave the for loop.
